# Documentation

This folder is reserved for user-facing documentation.

Start with the root [README](../README.md) for installation and the local quick start. Use the docs in this folder when you need more detail on one part of the system:

- [How it works](./HOW-IT-WORKS.md) explains the runtime model, response streaming, and where session state lives.
- [Phone access](./PHONE.md) explains inbound calling, dispatch setup, and the operational requirements for telephony.
- [Costs and limits](./COSTS.md) explains the main LiveKit pricing lines and the quotas that matter for the default local-worker setup.

Internal planning notes, marketing assets, and implementation scratch docs do not belong in this folder.
