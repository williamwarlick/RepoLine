# RepoLine Branding

## Name

`RepoLine` works because it says exactly what the project is: a line into your repository.

It keeps both halves of the idea:

- `repo`: the local codebase Claude Code can actually work in
- `line`: a phone line, voice line, or direct connection

## Recommended logo workflow

For a polished final logo, use a two-step workflow instead of relying on a single image generator:

1. Generate concept directions with [Ideogram](https://ideogram.ai/).
2. Refine the winning concept as an editable SVG in [Adobe Illustrator Text to Vector Graphic](https://www.adobe.com/products/illustrator/icon-generator.html).

Why this stack:

- Ideogram is good for exploring wordmark and concept directions quickly.
- Illustrator is better for the final mark because the output is editable vector artwork that can be exported cleanly as SVG.
- For open source, editable vector output matters more than a one-shot image.

## Visual direction

The current repo banner and mark use this direction:

- dark background
- mint to cyan accent gradient
- phone receiver motif
- angle brackets to suggest code
- clean geometric shapes instead of mascot art

That is the right direction for this project. It feels technical, voice-native, and local-first without looking like a generic AI assistant brand.

## Prompt starter

Use this prompt as the starting point:

> Design a modern logo for "RepoLine", an open-source tool that lets you call your codebase by phone. Combine a phone line or handset motif with subtle code cues like angle brackets. Use a dark background, mint-to-cyan gradient accents, and a clean geometric style. Avoid mascots, 3D renders, and generic chatbot imagery. Make it feel developer-first, crisp, and suitable for an SVG wordmark and app icon.
